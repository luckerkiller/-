计提调账系统 · 产品架构图

AI Agent 智能交互 → 计提调账系统（业务接入 + 功能层）→ CS 核算记账 → SAP 总账 · 平行数据与对账（右侧独立运行）

AI Agent
🤖 用户通过自然语言与系统交互，AI Agent 理解意图并调用对应系统能力 大语言模型驱动 意图识别 + 参数提取 多轮对话
📤 智能上传
  • 「帮我上传这份计提表」
  • 自动解析文件字段 + 校验
  • 异常字段提示并引导修正
  • 确认后自动提交审批流
🔍 单据查询
  • 「查一下本月收入计提状态」
  • 「ADJ-2025-003 审批到哪了」
  • 自然语言转结构化查询条件
  • 结果摘要 + 跳转详情
✅ 审批辅助
  • 「我有几条待审批」
  • 推送待审批摘要与风险提示
  • 支持对话确认通过 / 驳回
  • 批量处理并记录操作日志
📊 数据问答
  • 「本月成本计提总额是多少」
  • 「上月调账差异原因汇总」
  • 跨单据聚合计算与分析
  • 输出结论 + 来源数据溯源
